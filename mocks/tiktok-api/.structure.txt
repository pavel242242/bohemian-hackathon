Configuration:
- Port: 3001 (configurable via PORT env var)
- Format: JSON (matching TikTok API)
- Dependencies: Only Express.js
- Status: Production-ready

Endpoints:
1. GET /open_api/v1.3/campaign/get/ (all campaigns + filters)
2. GET /open_api/v1.3/campaign/get/:campaign_id (single campaign)
3. GET /health (health check)

Quick Commands:
npm install          Install dependencies
npm start            Start server on port 3001
docker build -t tiktok-ads-mock . Build Docker image
docker run -p 3001:3001 tiktok-ads-mock:latest Run container
